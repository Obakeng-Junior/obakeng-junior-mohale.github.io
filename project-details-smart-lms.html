Smart Learning Management System

Mobile & Web Application Completed: 2024
Flutter ASP.NET Core Firebase SQL Server REST API Entity Framework

Advanced Learning Management System (LMS) combining a Flutter mobile application for students with an ASP.NET Core MVC admin dashboard for course management. The system features an AI-powered Smart Tutor, course enrollment, real-time progress tracking, and comprehensive learning analytics.

This project was designed to modernize traditional learning systems by providing personalized learning experiences through AI integration and real-time feedback mechanisms. The mobile app offers seamless learning on-the-go, while the admin dashboard provides powerful tools for educators to manage content and monitor student progress.

Key Features:

  • Student mobile app with course browsing and enrollment
  • AI-powered Smart Tutor for personalized learning guidance
  • Real-time progress tracking and analytics dashboard
  • Admin web dashboard for course and user management
  • Secure authentication and role-based access control
  • Assignment submission and grading system
  • Discussion forums for student collaboration
  • Push notifications for course updates
